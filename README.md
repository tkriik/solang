# Development

## System dependencies

## Build

    $ cargo build

## Run

    $ cargo run

## Run tests

    $ cargo test

## Test coverage

Dependencies:

  * [kcov](https://github.com/SimonKagstrom/kcov)
  * [jq](https://github.com/stedolan/jq)

Generate and open line coverage report:

    $ ./scripts/coverage.sh
